The result of a Contains evaluation — the question judged, the verdict, the Kookie pin, the schema version, and a timestamp. The question is a `Fact.SubjectFact` held putatively ([[putative-fact]]), in the seat named `Query`; the case type keeps a subjectset-valued question out of the seat. Complete at birth and replayable; not a bare boolean.
