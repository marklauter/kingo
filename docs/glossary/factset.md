The set of facts of a given relationship on a resource that a fact-to-subject-set rewrite walks to find resources to re-evaluate. The exactly-specified member is a resource ([[fact]]); the walk pairs each with the computed relationship. Renamed from *tupleset* (2026-07-21): "tuple" is the paper's word for the thing Kingo names [[fact]]. The same day's drift sweep corrected the walk's resolved members from *subjects* to *resources* per [[resource-fact-case]] — a semantic fix distinct from the rename.
